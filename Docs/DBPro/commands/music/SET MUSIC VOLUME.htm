SET MUSIC VOLUME

This command will set the volume at which the music is playing.

  Syntax 
SET MUSIC VOLUME Music Number,Volume
  Parameters 
Music Number
Integer
The music number 
Volume
Integer
A percentage of the standard volume of play ranging from 0 to 100

  Returns

This command does not return a value.

  Description

The default value is 100, as a percentage of the standard volume of play. A value of 50 means the music is playing at half the normal volume. The default value of 100 is the highest volume you can set.

  Example Code 
cls

load music "..\Media\Music\mp3\music misty room.mp3",1

play music 1

print "music volume ",music volume(1)

wait 1000*4

set music volume 1,50

print "music volume ",music volume(1)

wait 1000*4

stop music 1

end
