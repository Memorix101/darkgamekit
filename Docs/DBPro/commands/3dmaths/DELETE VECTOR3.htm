DELETE VECTOR3

This command will delete a vector.

  Syntax 
Return Integer=DELETE VECTOR3(Vector)
  Parameters 
Vector
Integer
vector

  Returns

  Description

This is defined as a three float vector.

  Example Code 
r=MAKE VECTOR3(1)

r=MAKE VECTOR3(2)

r=MAKE VECTOR3(3)

r=MAKE VECTOR3(4)

SET VECTOR3 1,0,0,0

SET VECTOR3 2,0,0,0

SET VECTOR3 3,0,0,0

SET VECTOR3 4,0,0,0

COPY VECTOR3 1,2

DIVIDE VECTOR3 1,10

HERMITE VECTOR3 1,1,2,3,4,0.5

IF IS EQUAL VECTOR3(1,2)=1

length#=LENGTH VECTOR3(1)

LINEAR INTERPOLATE VECTOR3 1,2,3,0.5

MAXIMIZE VECTOR3 1,2,3

MINIMIZE VECTOR3 1,2,3

MULTIPLY VECTOR3 1,100

NORMALIZE VECTOR3 1,2

SCALE VECTOR3 1,2,100

length#=SQUARED LENGTH VECTOR3(1)

SUBTRACT VECTOR3 1,2,3

x#=X VECTOR3(1)

y#=Y VECTOR3(1)

z#=Z VECTOR3(1)

ADD VECTOR3 1,2,3

f#=1 : gb#=1 : BCC VECTOR3 1,2,3,4,f#,gb#

CATMULLROM VECTOR3 1,1,2,3,4,0.5

CROSS PRODUCT VECTOR3 1,2,3

dotproduct#=DOT PRODUCT VECTOR3(1,2)

ENDIF

r=DELETE VECTOR3(1)

r=DELETE VECTOR3(2)

r=DELETE VECTOR3(3)

r=DELETE VECTOR3(4)
