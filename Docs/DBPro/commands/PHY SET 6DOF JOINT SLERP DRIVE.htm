PHY SET 6DOF JOINT SLERP DRIVE

Set the slerp drive on a joint

  Syntax 
PHY SET 6DOF JOINT SLERP DRIVE ID, type, spring#, damping#, forceLimit#

PHY SET 6DOF JOINT SLERP DRIVE ID, type
  Parameters 
ID
Integer
identification number of the joint 
type
Integer
type of the drive, only option for this is 0 at the moment 
spring#
Float
spring coefficient 
damping#
Float
damper coefficient 
forceLimit#
Float
maximum force ( or torque ) that the drive can exert

  Returns

  Example Code
