PHY SET FLUID EMITTER RANDOM POSITION

Random vector with values for each axis direction of the emitter orientation.

  Syntax 
PHY SET FLUID EMITTER RANDOM POSITION ID, x#, y#, z#
  Parameters 
ID
Integer
identification number of the fluid object 
X#
Float
X axis 
Y#
Float
Y axis 
Z#
Float
Z axis

  Returns

  Description

The values have to be positive and describe the maximal random particle displacement in each dimension. The z value describes the randomization in emission direction.
