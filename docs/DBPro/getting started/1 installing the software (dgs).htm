Installing The Software from CD

1. Insert the DISC into the Drive. If your computer has autoboot, wait for the splash screen to appear. If your computer does not have autoboot, go to My Computer, select the Device containing the Dark Basic Professional DISC, right click the item and select Autoplay.



2. From the Boot Menu Selection Screen, click Install Dark Basic Professional to install the software onto your computer.

3. When the installation is complete, you will find a Dark Basic Professional icon has been created on your desktop. To run the software, double click this icon.

Installing The Software from Downloadable

1. Follow the instructions from the email you will have received to download the installer to your computer.

2. Simply double click the installer executable to install the software onto your computer.

3. When the installation is complete, you will find a Dark Basic Professional icon has been created on your desktop. To run the software, double click this icon.

Activating Dark Basic Professional

The latest version of the software uses an online activation system to initially validate your software. This activation is performed onlt once and no further online access will be made by the software after this initial activation. You need access to the internet and an email address, plus you must ensure the firewall is configured to allow both inbound and outbound communication throughout the activation process. Upgrading Dark Basic Professional

When an upgrade becomes available, you can simply download and install it freely from our website. Keep checking the forums for news on the latest version, or check out the Support section of the Dark Basic Professional website. The upgrade installer only works when it detects an existing installation of Dark Basic Professional, so if your registry has been altered and your installation path has been lost, you will need to re-install the software from your original DISC or installer file before applying the upgrade. If you un-install without manually clearing the "Dark Basic Professional" folder in "Program Files", and quickly re-install you should not have to re-activate your software as the certificates created by the activation process should still be present on your computer.
