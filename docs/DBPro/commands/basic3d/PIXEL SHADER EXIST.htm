PIXEL SHADER EXIST

This command will return a value of one if the specified pixel shader exists, otherwise zero.

  Syntax 
Return Integer=PIXEL SHADER EXIST(PixelShader Number)
  Parameters 
PixelShader Number
Integer
This value is an integer number such as 1.

  Returns

This value is an integer number such as 1.
