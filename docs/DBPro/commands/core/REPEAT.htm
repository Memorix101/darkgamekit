REPEAT

These commands will define a loop that will repeat until the Condition becomes true.

  Syntax 
REPEAT
  Returns

This command does not return a value.

  Description

Unlike the WHILE command, the repeat loop allows the command sequence within your loop to run at least once before the Condition is checked. You are able to break from the loop at any time by using the EXIT command.

  Example Code 
REPEAT

PRINT "in a conditional loop"

UNTIL SPACEKEY()=1
