image


This command set allows the loading, creation, deletion and modification of images and textures, either from scratch or from a selection of supported file formats.

LOAD IMAGE : LOAD IMAGE Filename, Image Number
SAVE IMAGE : SAVE IMAGE Filename, Image Number
DELETE IMAGE : DELETE IMAGE Image Number
GET IMAGE : GET IMAGE Image Number, Left, Top, Right, Bottom
PASTE IMAGE : PASTE IMAGE Image Number, X, Y
SET IMAGE COLORKEY : SET IMAGE COLORKEY Red Value, Green Value, Blue Value


LOAD ICON : Return DWORD=LOAD ICON(Icon Filename)
DELETE ICON : DELETE ICON Icon Handle


IMAGE EXIST : Return Integer=IMAGE EXIST(Image Number)
IMAGE WIDTH : Return Integer=IMAGE WIDTH(Image Number)
IMAGE HEIGHT : Return Integer=IMAGE HEIGHT(Image Number)


OPEN IMAGEBLOCK : OPEN IMAGEBLOCK Filename, Mode
EXCLUDE FROM IMAGEBLOCK : EXCLUDE FROM IMAGEBLOCK Exclude Path
CLOSE IMAGEBLOCK : CLOSE IMAGEBLOCK
PERFORM CHECKLIST FOR IMAGEBLOCK FILES : PERFORM CHECKLIST FOR IMAGEBLOCK FILES
IMAGE FILE EXIST : Return Integer=IMAGE FILE EXIST(Filename)
