PHY SET CYLINDRICAL JOINT LOCAL ANCHOR

Set the local anchor.

  Syntax 
PHY SET CYLINDRICAL JOINT LOCAL ANCHOR ID, index, x#, y#, z#
  Parameters 
ID
Integer
identification number of the joint 
index
Integer
the index of the rigid body that is attached to the joint that is to have its local anchor modified, this can either be 0 or 1 
x#
Float
position on the X axis 
y#
Float
position on the Y axis 
z#
Float
position on the Z axis

  Returns

  Description

This command will set the local anchor for the specified rigid body attached to the cylindrical joint.
