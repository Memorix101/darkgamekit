PHY SET MATERIAL RESTITUTION

Set the restitution property of a material.

  Syntax 
PHY SET MATERIAL RESTITUTION ID, value#
  Parameters 
ID
Integer
ID number of the material 
value#
Float
the new level of restitution

  Returns

  Description

The restitution of an object is used to determine how energy is lost. The lower the restitution, the more energy is lost and less it will rebound from an impact whereas a higher level of restitution means less energy is lost and the more it will reflect from an impact.
