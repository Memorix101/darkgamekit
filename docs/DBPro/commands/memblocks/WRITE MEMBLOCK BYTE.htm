WRITE MEMBLOCK BYTE

This command will write a byte into the specified location of the memblock.

  Syntax 
WRITE MEMBLOCK BYTE Memblock Number, Position, Byte
  Parameters 
Memblock Number
Integer
The memblock number must exist or the command will fail 
Position
Integer
The Position value is specified in bytes 
Byte
Integer
The Byte value must be a value between 0 and 255

  Returns

This command does not return a value.

  Description

The memblock must exist or the command will fail. The Position value is specified in bytes. The Byte value must be a value between 0 and 255. The parameters must be specified using integer values.

  Example Code 
MemblockNumber=1

MAKE MEMBLOCK MemblockNumber,1024

WRITE MEMBLOCK BYTE MemblockNumber, 0,255

rem Display data

cls

print "MEMBLOCK EXPRESSION DATA"

print

if MEMBLOCK EXIST(MemblockNumber)=1

print "memblock:";MemblockNumber

print "exist:";MEMBLOCK EXIST(MemblockNumber)

print "ptr:";GET MEMBLOCK PTR(MemblockNumber)

print "size:";GET MEMBLOCK SIZE(MemblockNumber)

print "byte:";MEMBLOCK BYTE(MemblockNumber, 0)

endif

rem Delete memblocks

if MEMBLOCK EXIST(MemblockNumber)=1 then DELETE MEMBLOCK MemblockNumber

do

loop

end
